# site-index-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库本身不针对任何特定域名或具体网站，仅作为静态 HTML 资源的集合与索引。

## 项目简介

`site-index-hub` 旨在提供一个集中化的 HTML 页面存档与展示场所。  
你可以将各类独立页面、信息卡片、导航页或工具界面以静态 HTML 形式存放于此，并通过 GitHub Pages 或其他静态托管服务进行发布。

仓库不限定页面的用途与内容范围，但要求每个页面均为独立完整的 HTML 文件，便于直接访问与引用。

## 目录说明

```
.
├── index.html          # 仓库首页 / 入口索引（可选）
├── pages/              # 存放所有独立 HTML 页面的目录
│   ├── example.html    # 示例页面（可删除或替换）
│   └── ...
├── assets/             # 公共资源目录（CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md           # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件应保持独立，避免依赖外部资源（或统一引用 `assets/` 中的文件）。
- `assets/`：存放页面共用的样式、脚本、字体、图片等资源，建议按需组织子目录。
- `index.html`：可选，可作为仓库的导航入口或说明页。

## 页面归档说明

所有归档的 HTML 页面应遵循以下基本规范：

1. **独立性**：每个页面应尽量自包含，若引用外部资源需确保路径正确或使用完整 URL。
2. **命名清晰**：文件名应反映页面内容或用途，避免使用无意义数字或乱码。
3. **编码规范**：推荐使用 UTF-8 编码，并在 `<head>` 中声明 `charset`。
4. **无侵入性**：页面不应包含恶意脚本、跟踪代码或违反法律法规的内容。
5. **可访问性**：建议为页面添加基本的标题、描述与语义化标签。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面将通过提交 Pull Request 或直接推送完成。
- 欢迎贡献符合归档规范的 HTML 页面，但请确保内容不涉及侵权、色情、赌博、暴力或任何违法信息。
- 若发现页面无法正常访问、资源缺失或存在安全问题，请通过 Issues 反馈。
- 仓库保留对不合规内容进行删除或修改的权利。

## 许可

本仓库默认不附加特定许可证。  
各页面作者可自行在页面内声明版权信息或使用协议。  
贡献者提交内容即视为同意在仓库内公开存档。
